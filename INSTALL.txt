# Requirements
Requires Python 3.4+, Spring Boot 1+, JDK 8+, sql2o 1.4+

# Linux steps

1) Download the project from https://github.com/MarcoSulla/ewa/archive/master.zip
2) extract the files where you want
2) open a shell
3) go to the extracted folder
4) > cd ewa-master
3) > sudo apt install python3-pip
4) if you want, you can create a virtualenv
5) > pip3 install -r requirements.txt
6) See https://marcosulla.github.io/ewa/#use for using it.

Bye!
