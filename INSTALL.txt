#INSTALL EWA

1) scarica il progetto da https://github.com/MarcoSulla/ewa/
2) apri un terminale e vai nella cartella del progetto
3) sudo apt install python3-pip
4) pip3 install -r packages.txt
5) cd src/
6) configurare un .ini definendo db e tabella da importare, prendendo come 
   esempio config_example.ini
7) python3 ewa.py --config NOMEFILE.ini
